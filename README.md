This repository contains a Weather Forecast Web Application built using HTML, CSS, and JavaScript.
The application fetches real-time weather data and displays current weather conditions through a clean and responsive user interface.


 Project Overview
The Weather App allows users to search for any city and instantly view:
Current temperature
Weather conditions
Additional atmospheric details

The project demonstrates API integration, DOM manipulation, and responsive UI design, making it a strong beginner-to-intermediate frontend project.

 Key Features
 Search weather by city name
 Real-time weather data
 Fully responsive UI
 Clean and modern design
 Fast and lightweight


 Tech Stack
HTML5
CSS3
JavaScript (ES6)
Weather API
Node.js (for package management)



 How It Works
User enters a city name
JavaScript sends a request to the Weather API
API returns live weather data
Data is dynamically rendered on the UI
Weather icons update based on conditions
